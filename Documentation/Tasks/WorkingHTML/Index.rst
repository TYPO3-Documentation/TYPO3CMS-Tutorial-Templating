Working with the HTML template
""""""""""""""""""""""""""""""

In this chapter we will prepare our HTML template to be ready to
interact with the TYPO3 CMS templating engine. The first part
explains the concepts of marks and subparts which are central to
TYPO3 CMS templating. In the second part we will work on the HTML
template itself.
